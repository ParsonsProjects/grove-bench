# Keyboard Shortcuts

## Tab Navigation

| Shortcut | Action |
|----------|--------|
| `Alt+1` | Switch to Activity tab |
| `Alt+2` | Switch to Changes tab |
| `Alt+3` | Switch to Checkpoints tab |
| `Alt+4` | Switch to Terminal tab |

## Search & Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+F` | Search in Activity tab |
| `Ctrl+R` | Open session finder (quick-switch between sessions) |

## Agent Controls

| Shortcut | Action |
|----------|--------|
| `Alt+M` | Cycle agent mode (Code / Plan / Edit) |
| `Alt+T` | Toggle extended thinking on/off |

## General

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line in prompt editor |
| `Escape` | Close panel / Stop agent |
